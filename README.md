# Danish Iryandy Portfolio Website

A lightweight personal portfolio and resume website for Danish Iryandy, built with HTML and CSS and hosted through ApartmentLab, my self-hosted homelab environment.

This website showcases my professional experience, technical skills, education, selected projects, and contact details. It is designed to act as both an online resume and a live demonstration of my ability to build, maintain, and deploy a clean static web application.

## Live Website

[https://danishiryandy.apartmentlab.org](https://danishiryandy.apartmentlab.org)

## About This Project

This repository contains the source code for my personal portfolio website. The website is intentionally kept simple, fast, and maintainable, using only static HTML and CSS.

The project is also part of my larger ApartmentLab homelab setup, where application code is kept separate from the infrastructure repository. During deployment, this repository is pulled into the ApartmentLab environment and served through a self-hosted web stack.

## Features

- Responsive personal resume website
- Light and dark mode toggle
- Professional experience section
- Technical skills section
- Project showcase section
- Education section
- Contact links
- Downloadable resume
- Mobile-friendly layout
- Static HTML and CSS only, with no frontend framework required

## Tech Stack

- HTML
- CSS
- GitHub
- Docker
- Nginx
- ApartmentLab self-hosted deployment
